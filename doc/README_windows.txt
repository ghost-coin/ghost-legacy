Ghost Core

Intro
-----
Ghost is an open source, decentralized privacy platform
built for global person to person eCommerce.

For more information, as well as an immediately useable, binary version of
the Ghost Core software, see https://ghost.io.


Setup
-----
Unpack the files into a directory and run ghost-qt.exe.

Ghost Core is the original Ghost client and it builds the backbone of the network.
However, it downloads and stores the entire history of Ghost transactions;
depending on the speed of your computer and network connection, the synchronization
process can take anywhere from a few hours to a day or more.

See the ghost wiki at:
  https://ghost.wiki/
for more help and information.
